How to read this diagram

This is an UpSet plot — a way to show Venn diagram intersections that works clearly even with many search phrases.

Each bar represents one region of the Venn diagram — a specific combination of phrases that co-occur in a passage.

Bar height = number of passages in that intersection.

Colored dots below each bar show which phrases contributed. A colored dot means that phrase matched; a grey dot means it did not.

Green bars = passages matching all your phrases (innermost Venn region).
Blue bars = passages matching some but not all phrases.

Click any bar to see the passages it represents in the list below.
Run a search to see the Venn diagram

Search phrase tips:

intracluster light — phrase match + synonyms + semantic

"coma cluster" — exact phrase only, no synonyms or semantic

NOT globular — put alone in its own box to exclude passages containing "globular" from all results

NOT "globular cluster" — exclude exact phrase

Synonym search: multi-word phrases look up the whole phrase as a synonym unit (e.g. "intracluster light" ↔ "ICL")

Semantic search: finds conceptually related passages even without exact word matches

Libraries

Libraries are separate bibliographic databases. Each library has its own papers, annotations, synonyms, and PDF files. Switching libraries changes the entire working context of BibMan. Libraries can be created at any time and deleted when no longer needed — deletion is permanent and irreversible.
